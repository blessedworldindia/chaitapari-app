# ChaiTapari

ChaiTapari is a social discussion and community platform where people can share thoughts, participate in conversations, and connect with others through posts and discussions.

## Features

* User Authentication
* Community Feed
* User Profiles
* Real-time Updates
* Firebase Backend
* Mobile Friendly Interface

## Installation

1. Download the latest APK from the Releases section.
2. Install the APK on your Android device.
3. Sign in and start using ChaiTapari.
